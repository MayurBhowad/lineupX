# LineupX

## Installation

Use the package manager [npm]() to install packages.

```bash
npm install
```

## .env

```python
MONGO_DB= [[database string]]
SESS_NAME = [[session name]],
NODE_ENV = development,
SESS_SECRET = [[jsonwebtoken secret]],
JWT_SECRET=[[jwt secret]],
```

## Running


```bash
npm run dev
```
or (production)
```bash
npm start
```
